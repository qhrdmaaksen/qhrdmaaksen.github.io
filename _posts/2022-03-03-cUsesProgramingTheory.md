1차원 배열 : int (data type) num(배열의 이름) [10] (데이터의 개수)
  int num[10];
2차원 배열 : int (data type) num(배열의 이름) [10] (행의 크기) [20] (열의 크기)
  int num[10][20]
배열의 크기 : int 4, char 1, double 8 byte
