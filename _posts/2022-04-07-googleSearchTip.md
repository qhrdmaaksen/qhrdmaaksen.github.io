## 검색할때 유용하게 써보자

```js

두 개 이상의 키워드를 검색할때는
  "청소" and "락스"
    "청소" or "락스"

키워드를 정확하게 모를때는
  페이스북 (검색을 원할때)
    페*북

특정 키워드를 제외 및 포함
   '-', 'intitle', 'intext' 등을 이용하면 더욱 효율적으로 검색을 할 수가 있다.

스타벅스 아이스 아메리카노
  위의 검색 결과중 스타벅스가 포함된 결과는 보고싶지않다면
    "아이스 아메리카노" - "스타벅스"
스타벅스가 포함된 키워드 검색
  "스타벅스"
    intitle:스타벅스
      여러개의 키워드가 포함된 검색을 원한다면
        "스타벅스" + "아이스 아메리카노"
          allintitle:스타벅스 아이스 아메리카노

본문에 특정 키워드 검색
  intext: 스타벅스

특정 웹사이트의 결과만 검색원할때
  site:"stackoverflow.com" Error

관련된 웹사이트를 함께 검색원할때
  related:google.com

특정 확장자 검색
  "마우스" filetype:pdf 

단어의 뜻만 찾기
  define: 거울

핸드폰 위치찾기
  find my phone

원하는 단어의 색 별로 검색
  검색창에 "헤어" -> 도구 색상 선택 

움직이는 짤 GIF 이미지 검색
  검색창에 머리 -> 도구 유형 애니메이션 선택

특정 날짜 기간 검색
  커피 2015..2017

URL 의 특정 검색어 검색
  inurl:mist




```
